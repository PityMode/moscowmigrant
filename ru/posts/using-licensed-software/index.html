Использование лицензированного софта

декабря 22, 2023 Читать около 3 минут
crime • russia • officials • problem • authority • politics • media • security • privacy

Из недавних эфиров Игоря Александровича Яковенко стало известно, что в России правоохранительные органы иногда используют обвинения в использовании нелицензионного программного обеспечения как предлог для конфискации компьютерной техники у журналистов. Хотя детали и даты этих случаев мне неизвестны, вопрос остается актуален. Это побудило меня рассказать о таком явлении, как FOSS (Free and Open-Source Software), которое играет важную роль в современном ИТ-мире.

Кратко о FOSS Link to heading

Программное обеспечение может распространяться либо с закрытым кодом (например, Windows, iOS, MS Office), либо с открытым кодом (Linux, Chromium, Libre Office). Закрытый код ограничивает понимание работы программы и использование ее кода в личных проектах. Обычно доходы от такого ПО идут за счет продажи лицензий, и любые попытки модификации или взлома являются нарушением закона.

Открытый код позволяет изучить и использовать программу в своих целях. Такие программы часто бесплатны, при этом доход формируется через сопутствующие сервисы, донаты и подписки.

Появление FOSS - это значимый шаг ИТ-индустрии. Мы получаем доступ к качественному софту, за который не нужно платить. Open-source часто предлагает более эффективные и безопасные решения благодаря открытости кода и международному сотрудничеству разработчиков.

Операционная система Link to heading

Выбор операционной системы - это первый шаг. Для серверов Linux является основным выбором, но для личных компьютеров ситуация иная: большинство использует Windows или MacOS. Но сегодня Linux доступен для всех пользователей, предлагая простую установку, современный интерфейс и широкий спектр программ.

Linux выделяется среди Windows и MacOS по нескольким параметрам:

  • Отсутствие телеметрии.
  • Высокая производительность и оптимизация.
  • Настраиваемые интерфейсы.
  • Возможность работы с флешки.

Для новичков хорошим выбором будет Linux Mint, а более опытные пользователи могут обратить внимание на Debian или EndeavourOS.

Интернет и Office Link to heading

Linux предлагает встроенные решения для интернета и офисных приложений. Большинство дистрибутивов поставляется с предустановленным браузером и комплектом офисных приложений, совместимых с форматами MS Office, особенно Libre Office.

Мессенджеры и другие приложения Link to heading

Большинство мессенджеров и приложений доступны для Linux или имеют альтернативы. Если программы нет для Linux, часто можно найти подходящую замена. Например, для замены Adobe Premiere Pro доступно множество альтернатив на Linux.

Составьте список программ, которые вы используете, и проверьте, доступны ли они на Linux. Если нет — найдите альтернативы. Вот список некоторых отличных FOSS приложений, которые вы можете исследовать:

  • Установка софта:
    • Для компьютера: Linux Mint, Debian, EndeavourOS, bauh, Webapp Manager.
    • Для смартфона: GrapheneOS, eOS, Aurora, Droid-ify.
  • Работа с документами:
    • Для компьютера: LibreOffice, Excalidraw, Joplin, Proton Calendar, Proton Drive, Collabora Office, MarkText.
    • Для смартфона: Joplin, Proton Calendar, Proton Drive, Collabora Office.
  • Социальные медиа и общение:
    • Для компьютера: FreeTube.
    • Для смартфона: Feeder, NewPipe SponsorBlock, AntennaPod.
  • Переписка и звонки:
    • Для компьютера: Element, Thunderbird, Proton Mail, Tutanota.
    • Для смартфора: Element, K-9 Mail, Proton Mail, Tutanota.
  • Обучение:
    • Для компьютера: GoldenDict, LanguageTool, Anki.
    • Для смартфона: AnkiDroid, GoldenDict, Librera, Openlib.
  • Интернет и навигация:
    • Для компьютера: Brave, LibreSpeed, ProtonVPN, Tor Browser, qBittorent.
    • Для смартфона: Brave, LibreSpeed, LibreTorrent, Organic Maps, ProtonVPN, Tor Browser, Vanadium.
  • Аудио и видео:
    • Для компьютера: mpv, Kdenlive, OBS Studio.
    • Для смартфона: mpv.
  • Безопасность:
    • Для компьютера: Bitwarden.
    • Для смартфона: Aegis, Bitwarden.

В заключении Link to heading

Переход с Windows на Linux и использование более 100 различных приложений не был для меня проблемой. Попробуйте и вы сделать шаг к светлому будущему, где ваша информация не используется в корыстных целях, а FOSS предоставляет свободу и безопасность.

FOSS — это замечательное явление, но оно требует нашей поддержки. Многие проекты держатся на нескольких разработчиках и нуждаются в финансовой поддержке. Помните, что ваш вклад — это поддержка уже сделанной работы, а не гарантия будущего обслуживания.
